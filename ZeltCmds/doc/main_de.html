Einleitung

ZeltCmds ist ein Befehlsplugin. Es hat keine feste Liste von Befehlen, nur einen Befehl zum verwalten des Plugins. Alles andere hängt von dir ab. Erstelle Befehle mit den Namen die du willst. Entferne Befehle die du nicht mehr brauchst. Dafür muss weder die Konfiguration neu geladen werden, noch der Server neu gestartet.

Es gibt 13 Kategorien, welche insgesamt 110 verschiedene Typen von Befehlen beinhalten. Tab-Vervollständigung ist für den Verwaltungsbefehl umgesetzt.

Einfache Verwaltung von Permissions durch Verwendung der Defaultpermissions. Permissionnodes stehen bereit für die erweiterte Verwaltung von Permissions.

Zusätzlich unterstützt ZeltCmds die Erstellung von Aliasen, welche nicht nur für plugineigene Befehle funktionieren. Aliase können in verschiedener Weise Parameter integrieren, mehrere Befehle abfertigen und die Ausführung verzögern.

Sprachschema für en und de verfügbar.

Kategorien

give
Gibt einem Spieler gegebene Gegenstände
mode
Ändert den Gamemode von Spielern
player
Spielerbezogene Befehle
serverinfo
Zeigt verschiedene serverbezogene Informationen
servertime
Setzen die Zeit für alle Welten des Servers
serverweather
Setzen das Wetter für alle Welten des Servers
set
Können Standorte für Bedspawns und Spawns von Welten setzen
spawn
Spawnt lebende Entitäten
sudo
Bringt jemand anders dazu einen Befehl auszuführen
teleport
Verschiedene Arten von Teleportbefehlen
worldinfo
Zeigt verschiedene weltbezogene Informationen
worldtime
Setzen die Zeit für bestimmte Welten
worldweather
Setzen das Wetter für bestimmte Welten

Änderungen der letzten Version

1.8.0

  • Kompiliert für Craftbukkit 1.6.2-R1.0
  • Neue Befehle in Kategorie player:
    • hide > Versteckt einen Spieler vor anderen
    • setflyspeed > Stellt die Fluggeschwindigkeit eines Spielers ein
    • setmaxair > Stellt die maximale Luft eines Spielers ein
    • setmaxhealth > Stellt das maximale Leben eines Spielers ein
    • settimeabsolute > Stellt die absolute Zeit eines Spielers ein
    • settimerelative > Stellt die relative Zeit eines Spielers ein
    • setwalkspeed > Stellt die Laufgeschwindigkeit eines Spielers ein
    • setweather > Stellt das Wetter eines Spielers ein
    • weather > Zeigt das Wetter für einen Spieler an
  • Neue Kategorie worldinfo mit Befehlen:
    • entity > Zeigt die Anzahl von Dingen einer Welt
    • gamerule > Listet Gamerules auf für eine Welt
    • info > Zeigt Informationen über eine Welt
    • livingentity > Zeigt die Anzahl von Lebewesen einer Welt
    • weather > Zeigt das Wetter einer Welt
  • Verschiedene interne Änderungen
  • Kleine Fehlerkorrekturen

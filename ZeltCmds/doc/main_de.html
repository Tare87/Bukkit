Einleitung

ZeltCmds ist ein Befehlsplugin. Es hat keine feste Liste von Befehlen, nur einen Befehl zum verwalten des Plugins. Alles andere hängt von dir ab. Erstelle Befehle mit den Namen die du willst. Entferne Befehle die du nicht mehr brauchst. Dafür muss weder die Konfiguration neu geladen werden, noch der Server neu gestartet.

Es gibt 10 Kategorien, welche insgesamt 67 verschiedene Typen von Befehlen beinhalten.

Einfache Verwaltung von Permissions durch Verwendung der Defaultpermissions. Permissionnodes stehen bereit für die erweiterte Verwaltung von Permissions.

Zusätzlich unterstützt ZeltCmds die Erstellung von Aliasen, welche nicht nur für plugineigene Befehle funktionieren. Kostante Parameter können dabei direkt im Alias integriert werden.

Sprachschema für en und de verfügbar.

Kategorien

mode
Ändert den Gamemode von Spielern
player
Spielerbezogene Befehle
serverinfo
Zeigt verschiedene serverbezogene Informationen
servertime
Setzen die Zeit für alle Welten des Servers
serverweather
Setzen das Wetter für alle Welten des Servers
set
Können Standorte für Bedspawns und Spawns von Welten setzen
spawn
Spawnt lebende Entitäten
teleport
Verschiedene Arten von Teleportbefehlen
worldtime
Setzen die Zeit für bestimmte Welten
worldweather
Setzen das Wetter für bestimmte Welten

Änderungen der letzen Version

1.4.0:

  • Kompiliert für Craftbukkit 1.4.5-R0.2
  • /zeltcmds list wird jetzt in Seiten aufgeteilt angezeigt
  • Alias-System hinzugefügt
    • Arbeitet allgemein, nicht nur mit Befehlen von ZeltCmds
    • Unterstützt Parameter innerhalb der Aliase
    • /zeltcmds addalias > Fügt neuen Alias hinzu
    • /zeltcmds listalias > Listet definierten Aliase auf
    • /zeltcmds removealias > Entfernt einen Alias
    • Mehr Informationen in der Referenz
  • Konfigurationsschlüssel logMsg umbenannt zu logCmd. Einstellung wird übernommen
  • Neue Befehle in Kategorie teleport:
    • m2e > Teleportiert den Spieler zur exakten Position
    • m2wc > Teleportiert den Spieler zum höchsten Block an den Koordinaten einer gegebenen Welt
    • m2we > Teleportiert den Spieler zur exakten Position einer gegebenen Welt
    • p2e > Teleportiert einen Spieler zur exakten Position
    • p2wc > Teleportiert einen Spieler zum höchsten Block an den Koordinaten einer gegebenen Welt
    • p2we > Teleportiert einen Spieler zur exakten Position einer gegebenen Welt
  • Kleine Fehlerkorrekturen

Changelog

1.6.0

  • Kompiliert für Craftbukkit 1.5.1-R0.1
  • Tab-Vervollständigung für /zeltcmds hinzugefügt
  • Konfigurationsschlüssel 'casesensitive' hinzugefügt für die Beachtung der Groß-/Kleinschreibung bei Befehlen von ZeltCmds
  • Neue Befehle in Kategorie teleport:
    • m2d > Teleportiert den Spieler zurück zu seinem letzten Todesort
    • m2l > Teleportiert den Spieler zurück zur letzten Position von der er teleportiert hat
    • mod > Teleportiert den Spieler in Blickrichtung über eine gegebene Distanz
    • p2d > Teleportiert einen Spieler zurück zu seinem letzten Todesort
    • p2l > Teleportiert einen Spieler zurück zur letzten Position von der er teleportiert hat
  • Kleine Fehlerkorrekturen

  • Kompiliert für Craftbukkit 1.4.6-R0.3

  • Alias-System kann mehrere Befehle zusammenfassen (siehe Referenz)

1.4.0

  • Kompiliert für Craftbukkit 1.4.5-R0.2
  • /zeltcmds list wird jetzt in Seiten aufgeteilt angezeigt
  • Alias-System hinzugefügt
    • Arbeitet allgemein, nicht nur mit Befehlen von ZeltCmds
    • Unterstützt Parameter innerhalb der Aliase
    • /zeltcmds addalias > Fügt neuen Alias hinzu
    • /zeltcmds listalias > Listet definierten Aliase auf
    • /zeltcmds removealias > Entfernt einen Alias
    • Beispiele in der Referenz
  • Konfigurationsschlüssel logMsg umbenannt zu logCmd. Einstellung wird übernommen
  • Neue Befehle in Kategorie teleport:
    • m2e > Teleportiert den Spieler zur exakten Position
    • m2wc > Teleportiert den Spieler zum höchsten Block an den Koordinaten einer gegebenen Welt
    • m2we > Teleportiert den Spieler zur exakten Position einer gegebenen Welt
    • p2e > Teleportiert einen Spieler zur exakten Position
    • p2wc > Teleportiert einen Spieler zum höchsten Block an den Koordinaten einer gegebenen Welt
    • p2we > Teleportiert einen Spieler zur exakten Position einer gegebenen Welt
  • Kleine Fehlerkorrekturen

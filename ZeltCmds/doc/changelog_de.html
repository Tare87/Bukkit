Changelog

1.7.0

  • Kompiliert für Craftbukkit 1.5.2-R1.0
  • Konfiguration für Update-Check separiert
  • Berücksichtig andere Plugins bezüglich reservierter Befehle
  • Verfügbare Parameter zum Aliassystem hinzugefügt:
    • <paramX> - Der Xte übergebene Parameter
    • <param?> - Ein optionaler Parameter
    • <param*> - Alle übergebenen Parameter die noch keiner Position zugewiesen bekommen haben
    • <player_name> - Name des ausführenden Spielers
    • <player_nick> - Anzeigename des ausführenden Spielers
    • <player_world> - Weltname des ausführenden Spielers
    • <player_x> - x-Koordinate des ausführenden Spielers
    • <player_y> - y-Koordinate des ausführenden Spielers
    • <player_z> - z-Koordinate des ausführenden Spielers
    • <item_name> - Name des Gegenstandes in der Hand des ausführenden Spielers
    • <item_id> - ID des Gegenstandes in der Hand des ausführenden Spielers
    • <item_data> - Data des Gegenstandes in der Hand des ausführenden Spielers
    • <waitX> - Warte X Ticks vor der Ausführung des Befehls
  • Neue Befehle in Kategorie player:
    • alwaysfly > Lässt einen Spieler fliegen, ohne landen
    • build > Lässt einen Spieler bauen, ohne Interaktionen
    • cleanchat > Säuber den Chat eines Spielers
    • enchanting > Öffnet ein Enchantingfenster für einen Spieler
    • enderchest > Öffnet die Enderchest für einen Spieler
    • fly > Lässt einen Spieler fliegen
    • freeze > Friert einen Spieler ein
    • ip > Zeigt die IP eines Spielers
    • item > Zeigt Informationen über das Item in der Hand des Spielers
    • leveldown > Senkt das Level eines Spielers um eins
    • levelreset > Setzt das Level eines Spielers auf null
    • levelup > Erhöht das Level eines Spielers um eins
    • mute > Lässt einen Spieler verstummen
    • starve > Lässt einen Spieler hungern
    • workbench > Öffnet eine Werkbank für einen Spieler
  • Neuer Befehl in Kategorie serverinfo:
    • ram > Zeigt Informationen über den RAM des Servers
  • Neue Kategorie give mit Befehlen:
    • chest > Gibt Items die eine Anzahl von Truhen füllen
    • item > Gibt eine bestimmte Menge von einem Item
    • stack > Gibt eine bestimmte Menge Stacks von einem Item
  • Neue Kategorie sudo mit Befehlen:
    • all > Lässt alle Onlinespieler einen gegebenen Befehl ausführen
    • console > Führt einen gegebenen Befehl auf der Konsole aus
    • player > Lässt einen gegebenen Spieler einen gegebenen Befehl ausführen
    • world > Lässt alle Onlinespieler einer gegebenen Welt einen gegebenen Befehl ausführen
  • Informationen zu Kategorie player, Typ info hinzugefügt
  • Verschiedene interne Änderungen

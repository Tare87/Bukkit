Introduction

ZeltCmds is a commandplugin. It has no hardcoded list of commands, but the one command to manage the plugin. Everything else is up to you. Create commands with the name you want. Delete commands you don't need anymore. Changing commands need no restart or reload.

There are 10 categories, containing 67 different types of commands in total.

Simple management of permissions by using defaultpermissions. Permissionnodes are provided as well for advanced management.

In addition, ZeltCmds supports aliases, which not only work with plugin related commands. Constant parameters can be integrated to aliases.

Localization for en and de avaiable.

Categories

mode
Changes gamemode of players
player
Playerrelated commands
serverinfo
Gives serverrelated informations
servertime
Sets time for the server
serverweather
Sets weather for the server
set
Can set location of bedspawn, or spawn of a world
spawn
Spawns living entities
teleport
Several types of teleports
worldtime
Sets time for a world
worldweather
Sets weather for a world

Changes of last version

1.4.0:

  • Compiled for Craftbukkit 1.4.5-R0.2
  • /zeltcmds list is now splitted in pages
  • alias-system added
    • Works in general, not only with commands of ZeltCmds
    • Supports parameters within aliases
    • /zeltcmds addalias > Adds a new alias
    • /zeltcmds listalias > Lists defined aliases
    • /zeltcmds removealias > Removes a alias
    • More informations at reference
  • Configurationkey logMsg renamed to logCmd. Doesnt reset config
  • New commands in category teleport:
    • m2e > Teleports the player to exact position
    • m2wc > Teleports the player to highest block at coordinates on given world
    • m2we > Teleports the player to exact position on given world
    • p2e > Teleports a player to exact position
    • p2wc > Teleports a player to highest block at coordinates on given world
    • p2we > Teleports a player to exact position on given world
  • Minor bugfixes

For changes of older versions, see changelog

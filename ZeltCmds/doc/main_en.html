Introduction

ZeltCmds is a commandplugin. It has no hardcoded list of commands, but the one command to manage the plugin. Everything else is up to you. Create commands with the name you want. Delete commands you don't need anymore. Changing commands need no restart or reload.

There are 13 categories, containing 110 different types of commands in total. Tab-Completion is implemented for managing command.

Simple management of permissions by using defaultpermissions. Permissionnodes are provided as well for advanced management.

In addition, ZeltCmds supports aliases, which not only work with plugin related commands. Aliases can integrate parameter in various ways, handles multiple commands and can wait before execution

Localization for en and de avaiable.

For more details see the reference


Categories

Gives specified items to player
mode
Changes gamemode of players
player
Playerrelated commands
serverinfo
Gives serverrelated informations
servertime
Sets time for the server
serverweather
Sets weather for the server
set
Can set location of bedspawn, or spawn of a world
spawn
Spawns living entities
sudo
Make someone else execute a command
teleport
Several types of teleports
worldinfo
Gives worldrelated informations
worldtime
Sets time for a world
worldweather
Sets weather for a world

Changes of last version

1.8.0

  • Compiled for Craftbukkit 1.6.2-R1.0
  • New commands in category player:
    • hide > Hides a player from others
    • setflyspeed > Sets or resets flying speed of a player
    • setmaxair > Sets or resets maximum air of a player
    • setmaxhealth > Sets or resets maximum health of a player
    • settimeabsolute > Sets or resets absolute time of a player
    • settimerelative > Sets or resets relative time of a player
    • setwalkspeed > Sets or resets walking speed of a player
    • setweather > Sets or resets weather of a player
    • weather > Shows the weather of a player
  • Added category worldinfo with commands:
    • entity > Lists the count of entities of a world
    • gamerule > Lists the gamerules of a world
    • info > Shows informations about a world
    • livingentity > Lists the count of living entities of a world
    • weather > Shows weather of a world
  • Several internal changes
  • Minor bugfixes

For changes of older versions, see changelog

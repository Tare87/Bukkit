Introduction

ZeltCmds is a commandplugin. It has no hardcoded list of commands, but the one command to manage the plugin. Everything else is up to you. Create commands with the name you want. Delete commands you don't need anymore. Changing commands need no restart or reload.

There are 10 categories, containing 73 different types of commands in total. Tab-Completion is implemented for managing command.

Simple management of permissions by using defaultpermissions. Permissionnodes are provided as well for advanced management.

In addition, ZeltCmds supports aliases, which not only work with plugin related commands. Constant parameters can be integrated to aliases. Multiple commands can be combined in one alias

Localization for en and de avaiable.

Several types of teleports
worldtime
Sets time for a world
worldweather
Sets weather for a world

Changes of last version

1.6.0:

  • Compiled for Craftbukkit 1.5.1-R0.1
  • Added Tab-Completion for /zeltcmds
  • Added configurationkey 'casesensitive' for handling commands of ZeltCmds casesensitive or not
  • New commands in category teleport:
    • m2d > Teleports the player to the location of last death
    • m2l > Teleports the player back to the last location ported from
    • mod > Teleports the player forward over a given distance
    • p2d > Teleports a player to the location of last death
    • p2l > Teleports a player back to the last location ported from
  • Minor bugfixes

For changes of older versions, see changelog

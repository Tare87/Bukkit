Introduction

ZeltCmds is a commandplugin. It has no hardcoded list of commands, but the one command to manage the plugin. Everything else is up to you. Create commands with the name you want. Delete commands you don't need anymore. Changing commands need no restart or reload.

There are 10 categories, containing 68 different types of commands in total.

Simple management of permissions by using defaultpermissions. Permissionnodes are provided as well for advanced management.

In addition, ZeltCmds supports aliases, which not only work with plugin related commands. Constant parameters can be integrated to aliases. Multiple commands can be combined in one alias

Localization for en and de avaiable.
